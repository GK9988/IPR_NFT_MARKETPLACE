# NFT Marketplace

## Setting Up

### 1. Clone/Download the Repository

### 2. Install Dependencies:

```
$ cd nft_marketplace
$ npm install
```

### 3. Boot up local development blockchain

```
$ cd nft_marketplace
$ npx hardhat node
```

### 4. Migrate Smart Contracts

`npx hardhat run src/backend/scripts/deploy.js --network localhost`

### 5. Run Tests

`$ npx hardhat test`

### 6. Launch Frontend

`$ npm run start`
